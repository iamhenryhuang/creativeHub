# CreativeHub

一個現代化的 React 創作社群平台，支援專案發佈、個人檔案編輯、協作者展示與即時互動。

## 快速開始

```bash
npm install
npm start
```

瀏覽器開啟 http://localhost:3000

## 主要功能
- 專案動態牆與發佈
- 個人檔案編輯（姓名、頭像、簡介、背景）
- 協作者展示
- 響應式設計，手機/桌機皆適用
- 搜尋與即時互動
